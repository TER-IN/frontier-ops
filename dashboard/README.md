# Frontier Operations Dashboard

This is the Next.js visual frontend for the **Frontier Operations** project. 

It reads the extracted claims generated by the backend Python pipeline and visualizes them on a dynamic dashboard. By aggregating the scattered signals from across the open web, this dashboard acts as a real-time map of the "Expanding AI Bubble."

## Architecture

This frontend is completely detached from the scraping/extraction pipeline. It relies exclusively on static data files generated by the backend:
1. `../data/CAPABILITIES.yml`: The internal taxonomy of IT and Engineering skills we are tracking.
2. `../data/extracted_claims.json`: The raw JSON output of all claims extracted by the LLM from our monitored internet sensors.

These files are read at request time (or build time in production) using Node `fs` in `src/lib/data.ts`.

## Development

First, ensure you have generated the baseline data by running the Python pipeline in the root directory:
```bash
# In the root repo directory
PYTHONPATH=. python src/main.py
```

Then, start the Next.js development server:
```bash
cd dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS V4
* **Icons:** Lucide React
* **Data Parsing:** `js-yaml`
